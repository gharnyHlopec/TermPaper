для запуска требуется установленный maven
1)для запуска войдите в корневую папку проекта через командную строку при помощи команды cd "путь к корневой папке проекта"
2)далее соберите проект при помощи команды mvn clean install
3)через командную строку перейдите в папку target (cd "путь к корневой папке проекта"\target)
4)запустите команду java -jar camelapp-0.0.1-SNAPSHOT.jar
5)перейдите по URL http://localhost:8080/ecommapp/order/process для получения ответа от сервера
